Project Summary

Chaos Compiler is a fast CoreXY-based printer with a "Cartesian-ish" Z system (single-motor lead screw with robust anti-wobble supports), automatic multicolor swapping via a homebrew MMU, and a servo-powered auto-eject system that physically kicks prints off the glass build plate. Built in India on a tight budget and driven by Klipper running on a Raspberry Pi 4.

Mechanics & Electronics

  • CoreXY motion for high-speed XY — paired with light carriage and precise pulleys.
  • Z axis: single motor with 8mm rods + custom gantry plates and anti-wobble supports.
  • Build plate: 310×310 glass with hairspray adhesion trick used during prints.
  • Controller: SKR MINI E3 V3 + Raspberry Pi 4 running Klipper.
  • MMU: servo pushes selection rod; a stepper drives the extruder stack. Filament sensors for no-buffer system.
  • Auto-eject: separate servo that physically pushes printed part off the bed.

Cooling & Structure

Hotend has a front-facing blower fan and side cooling for the heatsink. Frame uses 2020 aluminum extrusion for minimal wobble and rigidity; linear HGR15 rails for smooth XY motion.
